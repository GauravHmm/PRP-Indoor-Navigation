VIT PRP Academic Block Navigation

Navigate a sprawling 5-block academic complex with intelligent pathfinding. The system models the building as a graph with rooms, corridors, stairs, and elevators as nodes, finding optimal routes using the A* algorithm and Dijkstra's algorithm for nearest-facility search.

Next.js TypeScript A* Algorithm SVG Maps Multi-Floor

Key Features

📍

Interactive Maps

Zoomable, pannable SVG floor plans with color-coded rooms, corridors, and facilities. Real-time path overlay.

⚡

Smart Routing

A* pathfinding calculates optimal routes across multiple floors with distance-based heuristics and floor change penalties.

🔍

Instant Search

Find any classroom, lab, office, or facility with full-text search. Auto-find nearest washroom, water, or elevator.
